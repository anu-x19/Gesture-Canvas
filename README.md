# Gesture Canvas 🖐️

A real-time hand tracking AR web app that lets you draw on screen using hand gestures via webcam.

## 🔗 Live Demo
[Click here to view](https://anu-x19.github.io/Gesture-Canvas)

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- MediaPipe Hands (Google)

## ✨ Features
- Real-time hand tracking using webcam
- Draw on canvas using index finger
- Detects 21 hand landmarks
- Glassmorphism UI design

## 🚀 How to Run
1. Clone the repo
2. Open `index.html` with Live Server
